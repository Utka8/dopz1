Документация к проекту

Класс Program

Метод Main()
- Назначение: Главная точка входа в программу
- Логика:
  1. Разбивает исходную строку на слова по разделителям
  2. Выводит полученные части
  3. Собирает слова в обратном порядке
  4. Выводит результат

Используемые методы .NET
- string.Split() - разбиение строки по разделителям
- StringSplitOptions.RemoveEmptyEntries - удаление пустых элементов
- Конкатенация строк через оператор +=

Особенности реализации
- Обработка нескольких типов разделителей
- Автоматическое удаление пустых строк
- Обратный порядок сборки с разделителем " | "
